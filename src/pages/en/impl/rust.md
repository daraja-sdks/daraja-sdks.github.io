Documentation for the rust wrapper implementation of the `Daraja-sdks`.

## Installation
**On Rust version '^1.62':**
```bash
cargo add daraja
```
**On older versions of rust:**
```toml
[dependecies]
daraja="1.0.0"
(* ... *)
```
and then run...:
```bash
cargo check
```
